So benutzt du MediaGrab
Voraussetzungen – einmalige Installation
Bevor du MediaGrab startest, musst du zwei Dinge installieren. Öffne dafür die Eingabeaufforderung (Windows-Taste → cmd eingeben → Enter).
Dann tippe:
pip install customtkinter yt-dlp
Falls das nicht klappt, probier der Reihe nach diese Alternativen:
pip3 install customtkinter yt-dlp
python -m pip install customtkinter yt-dlp
py -m pip install customtkinter yt-dlp
Für MP3-Downloads brauchst du außerdem ffmpeg. Am einfachsten über den Windows Paketmanager:
winget install ffmpeg

Starten
Sobald alles installiert ist, einfach Doppelklick auf downloader.py – oder im Terminal:
python downloader.py

1. URL kopieren
Geh auf YouTube oder TikTok, such dir ein Video raus und kopier die URL oben aus der Adressleiste.
2. URL einfügen
Füg die URL in das Textfeld oben in MediaGrab ein.
3. Format wählen
Klick auf Video (MP4) wenn du das komplette Video willst – oder auf Nur Audio (MP3) wenn du z.B. Musik oder einen Podcast nur als Audiodatei brauchst.
4. Qualität einstellen
Beim Video kannst du zwischen best, 1080p, 720p, 480p und 360p wählen. best nimmt automatisch die höchste verfügbare Qualität.
5. Speicherort festlegen
Standardmäßig wird alles in deinen Downloads-Ordner gespeichert. Wenn du das ändern willst, klick auf Ändern und such dir einen Ordner aus.
6. Download starten
Klick auf den roten Download starten Button. Du siehst live wie weit der Download ist, wie schnell er lädt und wie lange er noch ungefähr dauert.
7. Fertig
Sobald der Download durch ist, steht unten grün Fertig! Gespeichert in: ... – die Datei liegt dann im gewählten Ordner.

Hinweis: MediaGrab funktioniert nicht nur mit YouTube & TikTok, sondern auch mit Instagram, Twitter/X, Reddit und über 1000 weiteren Seiten.
